# dockerize-flask-app
Run Flask app in Docker container

## To start Container

```
git clone https://github.com/Adiii717/dockerize-flask-app.git
cd dockerize-flask-app/
docker-compose up
```

## Publish Different port 

```
git clone https://github.com/Adiii717/dockerize-flask-app.git
cd dockerize-flask-app/
PORT=5001 docker-compose up
```



## Access Web Site

http://localhost:5000/
